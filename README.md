# Elipsometria
Projekt naukowo-badawczy wykorzystujący metody uczenia maszynowego (Deep Learning) do rozwiązywania problemu odwrotnego w elipsometrii spektroskopowej. System potrafi na podstawie widm elipsometrycznych ($\Psi$ i $\Delta$) automatycznie sklasyfikować rodzaj materiału oraz precyzyjnie wyznaczyć jego grubość.
